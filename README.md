# Text-Based Game (Python)

A simple command-line text-based adventure game built in Python.  
The player makes choices that affect the outcome of the story.

## 🎮 Features

- Interactive story-driven gameplay
- Multiple choices and branching paths
- Simple terminal-based interface
- Easy to modify and expand

## 🧠 How It Works

The game runs in the terminal. Players are prompted with choices and must type responses to progress through the story. Each decision affects the next scene or ending.

## 🛠️ Requirements

- Python 3.x

No external libraries required.
